# Incident Investigator AI

AI-assisted incident investigation workspace for multidisciplinary teams.

## What it does

Takes a short incident description plus four evidence types:
- Parte
- Posición
- Papel
- Persona

And produces an editable formal investigation report using a systemic analysis methodology.

## Initial scope

- Create investigations
- Upload and classify evidence
- Generate a first-pass systemic investigation draft
- Edit and refine the report collaboratively
- Export/share the final report
